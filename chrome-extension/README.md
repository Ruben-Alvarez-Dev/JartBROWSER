# Chrome Extension

JartBROWSER Chrome extension with sidebar, background service worker, and content scripts.

## Structure

```
chrome-extension/
├─ src/
│   ├─ background/
│   ├─ sidebar/
│   ├─ content-scripts/
│   ├─ offscreen/
│   └─ utils/
├─ manifest.json
└─ package.json
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```
