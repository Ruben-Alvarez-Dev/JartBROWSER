# Renderer

JartBROWSER React UI built with Vite and TypeScript.

## Structure

```
renderer/
├─ src/
│   ├─ components/
│   │   ├─ wizard/       # Setup wizard steps
│   │   ├─ dashboard/    # Post-install dashboard
│   │   ├─ settings/     # Settings panel
│   │   └─ common/       # Shared components
│   ├─ hooks/
│   ├─ stores/
│   ├─ App.tsx
│   └─ main.tsx
├─ index.html
├─ vite.config.ts
└─ package.json
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```
